# Reactor Foil

## Overview

The photobioreactor foil is a multi-layer disposable film assembly produced on the custom reactor machine. Each reactor consists of multiple panels welded together from four distinct plastic film rolls.

## Roll Function in the Reactor Sandwich

The reactor is a three-layer sandwich creating two functional chambers:

| Layer | Roll | Function |
|-------|------|----------|
| Top (non-transparent) | B | Light barrier / structural top |
| Middle | A | Panel formation / cultivation surface |
| Bottom | A | Panel formation / cultivation surface |

Tubing layers between the main layers:
- **Roll C** tubing runs between top and middle layers (feed and air/CO2)
- **Roll E** tubing runs between middle and bottom layers (pneumatic)

## Panel Structure

Each panel is an individual cultivation chamber formed by welding Roll A material. A standard indoor production reactor has 35 panels, each holding approximately 5.6 L of culture (197 L total per reactor).

## Reactor Types

| Type | Panels | Volume | Use |
|------|--------|--------|-----|
| Single-panel | 1 | ~5.6 L | Lab testing |
| 25-panel | 25 | ~140 L | Pilot testing |
| 35-panel | 35 | ~197 L | Indoor production |
