# Waste Water Unit (WWU)

## Overview

The Waste Water Unit handles pH correction and disposal of process waste water from the production system. Waste water originates from reactor cleaning, medium changes, harvest centrate, and CIP (Clean-In-Place) procedures.

## Function

The WWU receives waste streams from the MPC units via the waste valve V7 (automatic valve, kv 28.9 m3/h, DN32) on each MPC. Its primary function is pH neutralization before discharge to the municipal sewer system.

## Waste Water Sources

| Source | Description | Typical pH |
|--------|-------------|------------|
| Reactor flush water | Tap water used during end-of-campaign cleaning | Near neutral |
| Acid wash effluent | 0.05 M HCl used for biofilm dissolution | Low (acidic) |
| Alkaline/oxidative wash | Cleaning agents (P3 Oxonia, Javel) | High (basic) |
| Harvest centrate | Supernatant from centrifugation | Variable |
| Ozone treatment water | Post-ozonation rinse water | Near neutral |

## Process

1. Waste water is collected from MPC waste valves
2. pH is measured and adjusted (neutralization)
3. Treated water is discharged to sewer

## Cleaning Agents Requiring Neutralization

| Agent | Type | Neutralization Notes |
|-------|------|---------------------|
| P3 Oxonia | Peroxyacetic acid | Decomposes quickly, minimal neutralization needed |
| Javel (bleach) | Sodium hypochlorite | Requires extensive rinsing; alkaline |
| HCl (0.05 M) | Acid wash | Acidic, requires base neutralization |
| SDBS | Surfactant | Used for contamination treatment |
| Ozone | Oxidizer | Decomposes to O2, minimal residual |

## Gaps
