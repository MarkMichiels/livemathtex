# Sensors and Actuators

## Overview

Each MPC unit is equipped with a standardized set of sensors and actuators to monitor and control the photobioreactor environment. This document describes the sensor and actuator layout within one MPC unit.

![MPC Sensor and Actuator Layout](../images/mpc_sensor_layout.svg)
*Sensors and actuators within one MPC unit.*

## Sensors

| Sensor | Measurement | Location |
|--------|-------------|----------|
| pH probe | Culture pH | Per reactor |
| Temperature sensor | Culture temperature | Per reactor |
| OD sensor | Optical density (biomass) | Per reactor |
| DO sensor | Dissolved oxygen | Per reactor |
| Level sensor | Tank level | Feed/harvest tanks |
| Flow sensor | Medium flow rate | Feed manifold |

## Actuators

| Actuator | Function | Location |
|----------|----------|----------|
| Feed valve | Medium delivery | Per reactor |
| Harvest valve | Culture removal | Per reactor |
| Air valve | Aeration control | Per reactor |
| CO2 valve | pH control | Per reactor |
| LED panels | Light delivery | Per reactor |
| Pump | Medium/culture transfer | Per header |

## Calibration

Sensors require periodic calibration to maintain accuracy. pH probes and OD sensors are the most frequently calibrated instruments.
