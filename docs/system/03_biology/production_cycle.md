# Production Cycle

## Overview

The production cycle covers the complete lifecycle of a batch from inoculation through cultivation to harvest and downstream processing. Each cycle runs within the MPC-controlled photobioreactor system.

![Production Cycle Timeline](../images/production_cycle.svg)
*Complete production cycle from inoculation to downstream processing.*

## Cycle Phases

| Phase | Duration | Description |
|-------|----------|-------------|
| Inoculation | Day 0 | Fresh culture introduced to reactor |
| Growth | Days 1-7 | Vegetative growth under green-stage conditions |
| Stress | Days 7-14 | Nitrogen depletion triggers astaxanthin accumulation |
| Harvest | Day 14+ | Culture pumped to HBU tanks for downstream processing |

## Gaps

- Exact durations per phase for each species
- Light intensity and photoperiod schedules per phase
- Nutrient feed profiles (N, P, Fe) per phase
- Cell density targets at phase transitions
- Decision criteria for harvest timing
