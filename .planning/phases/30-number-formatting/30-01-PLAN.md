# Phase 30: Number Formatting - Plan 01

**Feature:** Thousands separator formatting for numbers >= 1000 (ISS-039)

## Objective

Lower the threshold for thousands separator formatting from 10,000 to 1,000.
The formatting already exists (`\,` LaTeX thin space), just applied to wrong threshold.

## Current State

- Thousands separators work for numbers >= 10,000
- Output: `$x == 10\,000\ \text{L}$`
- Numbers 1,000-9,999 have no separator

## Tasks

### Task 1: Lower Threshold
**Type:** fix
**Files:** `src/livemathtex/engine/evaluator.py`

Change threshold in `_format_pint_quantity_latex()`:
- Line 723: `>= 10000` → `>= 1000`

Test cases:
- 999 → no separator
- 1000 → `1\,000`
- 9999 → `9\,999`
- 37824 → `37\,824`

### Task 2: Add Config Option (Optional)
**Type:** feat
**Files:** `src/livemathtex/config.py`, `src/livemathtex/engine/evaluator.py`

Add `thousands_separator: bool = True` config option.
Skip this if time constrained - threshold change is the core fix.

## Success Criteria

- [ ] Numbers >= 1000 display with `\,` separator
- [ ] All existing tests pass (510+ tests)
- [ ] ISS-039 test file produces expected output
